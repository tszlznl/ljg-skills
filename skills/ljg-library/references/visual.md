# 视觉规格 + 七轮踩坑

生成 HTML 前过一遍。这是 ljg-library 的视觉质量底线，每一条都是七轮迭代换来的。

## 浅色光学玻璃（继刚精确规格，写死在 template，勿动）

| 角色 | 值 |
|------|-----|
| 背景渐变 | `#eef1f7 → #e3e8f2` 冷调雾白（**不是死白**） |
| 玻璃卡面 | `rgba(255,255,255,0.55)` + `backdrop-filter:blur(40px)` |
| 高光边 | `1px solid rgba(255,255,255,0.85)`（玻璃反光边） |
| 柔投影 | `0 8px 32px rgba(31,41,72,0.12)`（让它悬空） |
| 正文 | `#1a1f2e` 近黑不死黑，对比足、久看不累 |
| 次要 | `#5b6472` 中性灰蓝 |

三层明度浮起：雾白底 → 玻璃面 → 文字。浅底配深字（纸书传统，最护眼）。

## 动态强调色（skill 灵魂特性）

`python3 assets/extract_color.py <封面>` 从封面提主色，注入 template 的 `{{ACCENT}}`。`--accent-soft/line/glow` 由 `color-mix` 自动派生。换书自动换色：红封→红卡、蓝封→蓝卡。

**克制**：大面积是雾白底 + 深墨正文（占八成），强调色只点两成——标签、思想图曲线、取景框关键词高亮、署名印。不要泛滥。

## 字体

| 用途 | 字体 |
|------|------|
| 取景框句 / 书名 / 副标 | Noto Serif SC（书卷气） |
| 标签 / block-label / chart / 机读字段 | JetBrains Mono |
| 作者名 / 思想解读正文 | Noto Sans SC |

## 自适应 + 两端对齐

- 卡片 `flow + height:auto`，渲染必须 `fullpage`——高度跟内容，底部不留白。
- 所有正文文本 `text-align:justify; text-justify:inter-ideograph`——中文字距均匀撑满宽度、右侧齐平，根治「左对齐右侧豁口」。
- 任何信息块都不许只占左半：身份区左封面右信息填满、思想区左图右文填满。

## 结构（极简两 block + 身份区）

```
身份区：封面(198×287) | 书名+英文+副标+标签+作者头像  ← 填满右侧
─────
取景框 block：标签 + 那句话（主句 + 配象，justify）
─────
核心思想 block：左 ASCII 图 + 右三段解读
─────
极简署名：印 李继刚（唯一框架墨水，data-ink）
```

无 header、无 footer 日期、无收藏编号——Tufte 最大化信息墨水比，框架装饰全砍。

## 七轮踩坑清单（务必避开）

1. **封面尺寸**：weread `s_` 前缀 = 70×100 缩略图，必糊。换 `t7_` = 285×411 高清，下载带 `Referer: https://weread.qq.com/`。
2. **头像 thumb 陷阱**：Wikimedia `/thumb/.../NNNpx-` 若该尺寸未缓存 → 返 HTML 错误页。用**原图路径**（去 /thumb/ 和尺寸段）+ **User-Agent**（缺 UA 被拦）。
3. **ASCII 曲线非等距**：等距 `╱`（恒 45°）必成直线。凸曲线散点按 `x=√y` 非等距（底疏顶密）。详见 extraction.md。
4. **不要纯黑/纯白**：纯黑无法分层、配亮字割眼；用冷雾白底 + 近黑字。
5. **右侧空白**：justify + 左图右文，别让任何 block 只占左半。
6. **颜色克制**：强调只两成，从封面动态提取，不写死。
7. **取景框是认知位移非摘要**：动作动词 + 具体意象，「原来不是 X，其实是 Y」，不是「本书讲了……」。
