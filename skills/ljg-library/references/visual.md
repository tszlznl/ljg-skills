# 视觉规格 + 踩坑

生成 HTML 前过一遍。这是 ljg-library 的视觉质量底线。卡 = 浅色光学玻璃卡身（写死、不动）+ 一块嵌入的 AI 生成插画（图解板，吉田诚治风格）。

## 浅色光学玻璃卡身（继刚精确规格，写死在 template，勿动）

| 角色 | 值 |
|------|-----|
| 背景渐变 | `#eef1f7 → #e6ebf3 → #e3e8f2` 冷调雾白（**不是死白**） |
| 玻璃卡面 | `rgba(255,255,255,0.55)` + `backdrop-filter:blur(40px) saturate(1.3)` |
| 高光边 | `1px solid rgba(255,255,255,0.85)` |
| 柔投影 | `0 8px 32px rgba(31,41,72,0.12)` + `0 40px 80px -32px …`（悬空） |
| 正文 | `#1a1f2e` 近黑不死黑 |
| 次要 | `#5b6472` 中性灰蓝 |

三层明度浮起：雾白底 → 玻璃面 → 文字。卡身整体浅色不动；图解板里塞的生成图自带背景（吉田诚治暖光场景），是卡里唯一的「重色块」，对比着反而好看（已验证）。

## 卡身动态强调色（从封面提取）

`python3 assets/extract_color.py <封面>` 提主色 → `{{ACCENT}}`，用于标签、英文行、取景框关键词高亮 `<span class="hl">`、署名印。换书自动换色。**这是卡身色，和图解板生成图的配色互不干扰。** 强调色优先挑暖色（棕 / 橙 / 绿）更配那块暖调插画。

## 字体

| 用途 | 字体 |
|------|------|
| 取景框主句 / 费曼讲解 / 书名 / 副标 | Noto Serif SC（书卷气） |
| 标签 / EN / block-label | JetBrains Mono |
| 作者名 | Noto Sans SC |

（图解里的文字是生成图自带的——吉田诚治场景内自然手写字，不归 CSS 管。）

## 结构（身份区 + 取景框 + 图解 + 署名）

```
身份区：封面(198×287) | 书名+英文+副标+标签+作者头像
─────
取景框 block：标签 + 主句 {{FRAME}} + 费曼讲解 {{EXP}}（把意向画面讲透，justify）
─────
图解 block：嵌入的 AI 生成插画 {{SKETCH_IMG}}（16:9，吉田诚治风格，继刚作主角）
─────
极简署名：印 李继刚
```

无 header、无 footer 日期、无收藏编号——Tufte 最大化信息墨水比。

## 图解板（block 2）—— 嵌入 AI 生成插画

图解板 `.plate` 现在是一个**裱框容器**：`border-radius:16px; overflow:hidden`，里面塞 16:9 的生成 PNG（`<img class="sketch">` 满宽）。图自带背景，板只给圆角 + 柔投影。CSS 已写死在 template。

**风格 DNA（写死在 `assets/gen_illustration.py`，frame 不用写风格词）= 吉田诚治式**：绘本感异世界日常空间、软厚涂；暖光斜射 + 浮尘；蜜糖/琥珀/木棕暖调 + 雾化青绿点缀；木梁书架、植物书本器物细节堆叠；建筑透视扎实；魔法工房/旧书店/图书馆场所感；场景内自然手写中文字；继刚=栖居其中的小身形人物。

**主角恒为继刚**：脚本把 `assets/ljg-portrait.png` 当 character reference 喂模型，出「认得出的他」（吉田诚治场景栖居版）。认得出即可，风格化、不追同一张脸。

完整生图流程（写 frame → 跑脚本 → 验收）见 `references/extraction.md` 第二部分；worked frame 范例见 `references/example.md`。

## 自适应 + 两端对齐

- 卡片 `flow + height:auto`，渲染必须 `fullpage`——高度跟内容，底部不留白。
- 取景框文本 `text-align:justify; text-justify:inter-ideograph`——右侧齐平。
- 图解板生成图 `width:100%`，16:9 自然撑满。

## 踩坑清单

1. **封面尺寸**：weread `s_` = 70×100 缩略图必糊。换 `t7_` = 285×411，下载带 `Referer: https://weread.qq.com/`。
2. **头像 thumb 陷阱**：Wikimedia `/thumb/.../NNNpx-` 未缓存返 HTML 错误页。用原图路径 + User-Agent。
3. **生图必看图**：非确定 + 偶尔糊中文 / 继刚不像 / 没画对——必 Read 生成图亲验，不行调 frame 重生（标注短而少、主角说清、隐喻具体最稳）。
4. **frame 要"可画"**：抽象命题画不出，落成可见的物 / 动作 / 对比（见 extraction.md）。
5. **两套色别串**：卡身强调色从封面提取（标签/高亮/印）；图解背景是生成图自带（吉田诚治暖光场景）。
6. **主角风格化不追同一脸**：从墨像参考生成、认得出即可。别再回头合成真墨像贴脸（已弃，生图风格化更自然）。
7. **/tmp 文件名竞争**：并行铸卡的生成图用唯一名（带书 slug），别共享固定名。
8. **批量**：`gen_illustration.py` 直调 marswave、不走 listenhub 交互门控，ljg-paper-flow 批量可用；但每张要联网+API+看图一回合。
