# 取景框提炼 + 作者核心思想图

这是 ljg-library 的命门。卡好看是壳；从一本书提炼出它独创的看世界方式、压成一句话，才是命。失手则整张卡退化成豆瓣读书卡。

## 一、取景框提炼（六步）

1. **问题**：这本书在回答什么问题？
2. **零点**：在它之前，人们 / 各流派怎么看这个问题（旧共识）？
3. **位移（delta）**：这本书把认知挪到哪？——新取景框就在这一步的落差里。
4. **压句**：把位移写成一句「原来不是 X，其实是 Y」的**换眼睛**句。认知位移，动作动词；**禁**名词定义、**禁**「本书讲了……」式摘要。
5. **配象**：给它一个具体意象 / 隐喻，让读者半年后能在脑中**重演**那个顿悟。
6. **校验**：删掉书名作者，这句能否独立成立为「一种看世界的方式」？是摘要 → 重炼；是位移 → 过。再过翻译腔五连问（主语 / 动词 / 介词 / 定语 / 朗读卡），确保 native。

> **验收尺**：不是「好看」，是「凉了的脑子，瞥一眼，那一格瞬间回不回得来」。

### 示例 ·《反脆弱》

- 问题：怎么面对不可预测的冲击？
- 零点：要么脆弱（怕冲击）、要么强韧（扛冲击）——大家默认反义词是「坚固」
- 位移：塔勒布造了语言里缺的第三态 antifragile——**从冲击中获益**
- 压句（{{FRAME}}）：脆弱的反义词不是坚固，而是 <span class="hl">反脆弱</span>。
- 配象（{{ECHO}}）：它从波动和混乱里长力气——风吹灭蜡烛，却把火越烧越旺。
- 校验：删掉书名，这句独立成立为一种看世界方式 ✓；朗读 native ✓

### 卡面切分

- `{{FRAME}}`：压句（主句）。关键词（如「反脆弱」）用 `<span class="hl">…</span>` 染强调色。主句 ≤18 中文字符能一行放下最好。
- `{{ECHO}}`：配象（隐喻句）。

## 二、作者核心思想 ASCII 图（{{CHART}} + {{NOTE}} + {{CHART_TITLE}}）

block 2 = 把作者的核心思想画成一张极简 ASCII 图。它是这本书思想的骨架，不是装饰。

### 选什么形

| 思想形态 | 图形 | 例 |
|---------|------|-----|
| 函数 / 曲线 | 画那条曲线 | 塔勒布凸性 → 向上开口凸曲线 |
| 层级 / 结构 | 金字塔 / 树 | 马斯洛需求 |
| 循环 / 反馈 | 环 + 箭头 | 飞轮 / 复利 |
| 对立 / 光谱 | 轴 / 光谱条 | 二元张力 |

选**能一眼传达该书思想骨架的最简形**。`{{CHART_TITLE}}` 是这张图的名字（如 `Convexity 凸性`）。

### 凸曲线画法 —— 务必非等距（踩了三次的坑）

**等距 `╱` 字符（恒 45°）= 常斜率 = 必成直线，画不出凸性。**

凸曲线（`y=x²`，向上开口：左下平 = 损失 capped，右上陡 = 收益 open）必须**散点按 `x=√y` 非等距定位**——纵轴每行一个点 ●，横向缩进 = √(该行高度) × k。底部水平跨度大（横铺、平缓），顶部水平跨度小（近垂直、加速）。

《反脆弱》的 `{{CHART}}` 实际值（可直接复用/参照，注意缩进是真空格，自底向顶逐渐近垂直）：

```html
<span class="lbl">收益</span>
<span class="ax">│</span>                   <span class="cv">●</span>
<span class="ax">│</span>                 <span class="cv">●</span>
<span class="ax">│</span>               <span class="cv">●</span>
<span class="ax">│</span>            <span class="cv">●</span>
<span class="ax">│</span>         <span class="cv">●</span>
<span class="ax">│</span>      <span class="cv">●</span>
<span class="ax">│</span>   <span class="cv">●</span>
<span class="ax">│</span><span class="cv">●━━</span>
<span class="ax">└───────────→</span> <span class="lbl">波动</span>
```

缩进序列（顶→底）：19, 17, 15, 12, 9, 6, 3, 0。**差值递增（2,2,3,3,3,3,3…自底向上递减）= 凸**。底部 `●━━` 是 capped 平台。

### 右侧解读 {{NOTE}}

3 段，每段：`.k` 标题（关键词用 `<b>` 染强调色）+ `.v` 一句白话。把图的思想说成人话。《反脆弱》示例：

```html
<div class="cn-item"><div class="k">下行<b>封死</b></div><div class="v">损失有底，亏得起的小注</div></div>
<div class="cn-item"><div class="k">上行<b>敞开</b></div><div class="v">收益无顶，赌罕见的巨大命中</div></div>
<div class="cn-item"><div class="k">偏离越大，越<b>获益</b></div><div class="v">波动和混乱，成了它的养料</div></div>
```
